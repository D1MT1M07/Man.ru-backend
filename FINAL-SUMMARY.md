## 💡 БЫСТРЫЕ ПРИМЕРЫ

### Фильтрация
```javascript
// Показать статьи о фитнесе
const fitness = dataManager.getArticlesByCategory('fitness');
Renderer.renderArticles(fitness);
```

### Добавление статьи
```javascript
dataManager.addArticle({
    title: 'Моя статья',
    category: 'tech',
    description: 'Описание',
    author: 'Я'
});
```

### Работа с форумом
```javascript
// Создать тему
dataManager.addForumPost({
    title: 'Вопрос',
    category: 'career',
    message: 'Текст',
    author: 'User'
});

// Получить все темы
const posts = dataManager.getAllForumPosts();

// Удалить тему
dataManager.deleteForumPost(postId);
```
