ФУНКЦИОНАЛЬНОСТЬ:
==================

Поиск:
------
- При нажатии в поле поиска на любой странице (кроме главной) будут показаны ТОЛЬКО 
  релевантные результаты для этой страницы
- На странице "Здоровье" - только статьи о здоровье
- На странице "Фитнес" - только статьи о фитнесе  
- На странице "Форум" - только посты форума
- Выпадающий список теперь компактнее (250px вместо 400px)

Подписчики:
-----------
- Пользователи могут подписываться друг на друга через страницу users.html
- В профиле каждого пользователя отображаются счётчики:
  * Статей написано
  * Постов на форуме
  * Подписчики
  * Подписки (на кого подписан этот пользователь)
- Подписки сохраняются в localStorage

КАК ИСПОЛЬЗОВАТЬ:
==================

1. Поиск по категориям:
   - Откройте любую страницу (здоровье, фитнес, форум)
   - Нажмите на поле поиска в хедере
   - Вводите запрос - будут показаны результаты только для этой страницы

2. Подписка на пользователей:
   - Откройте страницу users.html
   - Найдите интересующего пользователя (поиск работает по имени и email)
   - Нажмите кнопку "Подписаться"
   - На странице профиля будут обновлены счётчики

3. Просмотр профиля:
   - Откройте profile.html (доступно если авторизованы)
   - Увидите свою статистику включая подписок и подписчиков
